# Secret Messenger Console

This is a simple Python command-line application that allows you to:
- Encode secret messages using a numeric key
- Decode previously encoded messages using the same key
- View saved encoded messages stored in a local file

## Features

1. Encrypt a message using a secret key  
2. Decrypt the encrypted message  
3. View stored encrypted messages  
4. Exit the console  

## Installation
mkdir secret_messenger
cd secret_messenger
```

## Usage

Run the script using:

```bash
python3 secret_messenger.py
```

## Requirements
No external libraries required. Works with standard Python 3.

## File Storage
- Encoded messages are stored in `data/message.txt`
