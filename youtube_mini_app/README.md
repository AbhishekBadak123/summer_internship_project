\# YouTube Clone Project



A simple YouTube clone with \*\*Flask backend\*\*, \*\*PostgreSQL database\*\*, and \*\*NGINX frontend\*\*.  

The project allows fetching video metadata via an API and serves a static frontend.  



---



\## Features



\- Flask REST API for videos

\- PostgreSQL database to store video metadata

\- NGINX frontend to serve HTML

\- Dockerized setup for all components

\- Custom Docker network for container communication



---



\## Prerequisites



\- Docker installed on your machine

\- Basic knowledge of Docker commands
