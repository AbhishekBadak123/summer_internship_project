A simple Streamlit-based web app to convert files \*\*between PDF and Word formats\*\* with easy-to-use UI and instant download capability.



\## Features



\- \*\*Convert PDF to Word (.docx)\*\*

\- \*\*Convert Word (.docx) to PDF\*\*

\- File upload and download with progress indicators

\- Friendly UI with tabs for each conversion direction

\- Status notifications and error handling

\- Temporary file cleanup for security

\- Download files named with your name (`abhishek\_converted.\*`)



\## Installation



First, ensure you have Python 3.8+ installed.



Install dependencies:





pip install streamlit pdf2docx python-docx reportlab

```

\## Usage



Run the app with:



streamlit run app.py

```



1\. Open the local URL generated by Streamlit.

2\. Select the desired tab for conversion (`PDF --> Word` or `Word --> PDF`).

3\. Upload your file and click convert.

4\. Download the resulting converted file.
